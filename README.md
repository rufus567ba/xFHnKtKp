# 项目简介

## 核心功能
- 基于 YOLOv8 的害虫检测系统，支持 6 类害虫识别：粘虫、小地老虎、蛴螬、蝼蛄、桃蛀螟、红蜘蛛
- 提供 3899 张完整标注的数据集
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 支持批量检测功能，可在界面直接查看所有检测结果
- 支持检测结果保存
- 提供完整程序文件、UI界面源文件、图标及项目运行教程
- 支持自定义置信度和 IOU 阈值设置
- 支持选择显示标签和原图
- 支持选择检测模型

## 适用场景
- 农业害虫监测与识别
- 农作物病虫害防治研究
- 农业生产环境中的害虫检测
- 农业科研与教学演示

## 技术特点
- 基于 Python 3.9 开发，依赖环境包括 opencv-python、PyQt5、ultralytics、torch1.9 等
- 提供 Pycharm+Anaconda 或 VSCode+Anaconda 两种开发环境选择
- 采用 YOLOv8 作为核心检测算法，具有较高的检测精度和效率
- 提供图形化用户界面，操作简便直观
